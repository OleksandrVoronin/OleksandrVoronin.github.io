2017 - Android Game Development Intern

During my 2017 co-op term, working as an Android game developer I had a chance to work on many different games, using Unity engine, coding in C#. My responsibilities also included porting and building some of the games to iOS. 

I was developing both 2D and 3D games, handling game design, all the gameplay programming, optimization, and QA. I had a chance to work with network programming using Photon Engine, shaders programming using HLSL and integration of various services such as Firebase Analytics, Admob ads, AppsFlyer, and Google Play Services. 

I gained a lot of experience with 3D modeling, animation and Unity Mecanim system for creating animation state-controllers, Later in my co-op term, I had a chance to tutor and supervise newcomers to our team and teach them some of the things I've learned through this co-op term.
