2016 - Amber

Amber is a fast-paced platformer written in C# using Unity engine. This is my first major experience with this engine outside of some assignments I've written in Coursera game-development courses. ​ 
This project was abandoned, but many cool features such as dynamic lighting 2D, skills system, modular AI and save-load checkpoints were implemented. Working on this project was a great experience that allowed me to familiarize myself with an engine, animation tools, shaders, debugging and profiling tools; as well as practice C# in this environment.

2015 - Robot Farm

Robot Farm is an open world RPG developed together with the NokoriWare team (http://www.nokoriware.com/). 
Written in Java, using LWJGL as a framework to take a full advantage of OpenGL rendering, Robot Farm is classic RPG inspired by games such as Harvest Moon and Pokemon. With a procedurally generated world, hundreds of skills and compelling story, this game brings lots of replayability and hours of entertainment. 
Working on this project my responsibilities included developing complex AI and a turn-based combat system along with balancing hundreds of skills and traits in the game. During this project, I gained experience of working in a team and using GitHub for distributed development.

2014 - Fireflies

Fireflies is a simple real-time strategy fully written in Java; the .awt library is used to render the game. 
The game includes an advanced AI with a decision-making system that evaluates the current in-game situation from both players’ perspectives in order to choose a right action in a given situation. Due to a high number of units present in the game at the same time, the AI also had to be well optimized and separated into separate modules - governors, each responsible for a single planet, yet all acting cohesively. 
During this project I’ve learned a lot about game design, inheritance and modularity in programming; developed an advanced AI and a simple particle system.

2013 - Lights Out

Lights Out is an unfinished top-down shooter game inspired by Hotline Miami. Written in Java, using java.awt library for rendering, it was a great experience in developing a fast-paced action game where smooth and lag-free experience is essential. ​ During this project I’ve learned a lot about pathfinding, ray tracing techniques developed a basic AI, lighting and particle system for blood splashes. ​ Due to lack of interest in content creation and visual aspects, I did not finish the game, although all major systems were coded and I have a somewhat playable demo.

2012 - Warp 2D

Warp was my first experience developing a game for Android OS, inspired by a Windows game called RoX by Safrosoft. Fully written in Java using standard Android/Java libraries, it was a great way to learn basics of game development, animation, and level design. Through trial and error, I managed to develop a finished product from scratch. This game has a number of flaws and trails of poor design such as poor memory management (which resulted in crashes on many Samsung devices) and bad usage of concepts of object-oriented programming (poor understanding of object inheritance led to a very messy code structure).
